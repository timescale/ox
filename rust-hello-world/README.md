# Rust Hello World

A simple "Hello, World!" application written in Rust.

## Prerequisites

- Rust toolchain (rustc and cargo)
- Install from: https://rustup.rs/

## Building

```bash
cargo build
```

## Running

```bash
cargo run
```

Or build and run the executable directly:

```bash
cargo build --release
./target/release/hello-world
```

## Project Structure

```
rust-hello-world/
├── Cargo.toml          # Project manifest
├── src/
│   └── main.rs        # Main source file
└── README.md          # This file
```

## Output

```
Hello, World!
```
